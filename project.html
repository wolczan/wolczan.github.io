Sharepoint online script

DLL files are related to SharePoint Client Object Model, which allows interaction with SharePoint servers using code.

By loading DLL files, we are making the types and functionality defined in these libraries available for use in your PowerShell script. Once the DLL files are loaded, we can create objects, call methods, and access properties defined in these libraries to interact with SharePoint servers, manage user profiles, and perform various operations within the SharePoint environment.

  • three (Dynamic Link Library) files

    Microsoft.SharePoint.Client.UserProfiles.dll

    Microsoft.SharePoint.Client.dll

    Microsoft.SharePoint.Client.Runtime.dll
